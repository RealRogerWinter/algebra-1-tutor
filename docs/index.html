Algebra 1

A complete, self-paced Algebra 1 course for an adult who knows arithmetic: a friendly textbook to read and a one-on-one tutor to learn with, plus extra practice when you want it. 13 units and 53 lessons in all, from foundations to a quadratics capstone plus a core unit on data and statistics, each quotable by a short code.

  • Student guide
    Start here: how the course works and how to learn with the tutor.
  • Textbook
    The lessons in full — worked examples, practice, and 23 computed figures, each with a reference code you can deep-link or quote.
  • Tutor guide
    273 extra parallel-form problems with full worked solutions, plus a mixed-review set per unit.

See a code like 12.5.w2 beside a worked example? Quote it to the tutor — "explain 12.5.w2" — and it pulls up that exact item, re-checks the math, and walks you through it.

Math is rendered with KaTeX. The whole course is generated from one source of truth and held correct by sympy and a CI gate.
